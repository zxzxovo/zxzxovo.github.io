一阵风吹走了我捡起放在手上的落叶，日历上的2025年也来到了尽头。

有些东西会被它带走，风带走了我手上的落叶，也带走了未被捡起的落叶。它会带走弥漫的烟雾，带走一丝体温，带走飘渺在摆动围巾上的碎语。许多寻常的事物也跟风一样，如霓虹间穿梭的车辆，五号线兜圈的地铁，大桥下流淌的河流。它们像风一样在你面前经过，也同样悄悄偷走你的东西——或许只是它们渐渐淡去了，而我们只留意到路过的风。

这一年终于结束了。许多人将时间作为自己人生的坐标与参照，如若今天过得不是很好，就希望下一天能好好生活，而早早地远离今天了。好像如果时间离得越远，发生过的悲伤，尴尬的事也就会远离自己了。我并不希望今年早早过去，即使许多日子充满悲伤，我是盼着它之后的新的一年，我有着很多寄托在明天的希望。

在路上的行人不知不觉丢掉了许多东西，最后连自己都不知道还剩下什么。不要去回想吧，已经走了很远找不回来了。但如果自己也主动扔掉那些记忆，难道不是辜负了曾经它们对于我的意义？我手上的落叶与我尚未捡起的，还有已在随风飘荡的，它们同等重要。

我要怎么回顾我这一年？既然被叫做“做题家”，那么就完成一场自问自答的考试吧。

## Q

### 1. 用一个字，一个词语，一句话，描述一下你的2025.

答：
字：柔
词语：Resilient
一句话：心态、情绪、世界观和生活方式有比较大的变化，尝试了曾经害怕的一些事情，总的来说，这一年是我真正开始珍视生命和自由，从头开始认识这个世界的开始。


### 2. 聊聊现在你对自己的认识和了解吧，和之前有什么不同呢？

答：
我是一个来自社会中下层，资质平平，样貌普通，脆弱敏感，但也可能有些有趣的，希望能过好平凡的生活并在其中找到幸福和爱的小女孩（自以为OwO，其实已经是毒妇了）。
	
这一年经历了许多事情，我对自己认识的最大转变是，接受了自己是普通人。
从前我心底一直有种傲气，认为自己在某些方面出类拔萃。但在这一年里，当我不再只是纸上谈兵，而是按自己的想法实际做某些事时，我才真切的明白自己竟是如此平庸。
你能想象到我看一个小时书就能把自己看睡着吗，而且看的还是我非常喜欢的书？阅读吸收知识的速度也很慢，我得一字一句读完才能勉强把知识留在脑子里，不然刚进入脑子里的信息就会转瞬即逝。在获取信息和思考上我总会慢一拍，好像脑子里面没有任何一个正常的神经通路，想到一半就会断掉。真的能把自己气死，但也没办法，挺无奈的。
我还尝试过一些思维训练，以及ADHD的一些药物，有些效果但不明显。“很遗憾，你已经确诊了笨笨”，我之前好像不是这样的，但就算如此也回不去了。所以哈哈，自己的问题只能自己好好接受了，不然还能干嘛。
	
对于容貌焦虑，其实一直都有一些，在22年做完鼻中隔偏曲的手术后更严重了。因为动了鼻子前边的一些软骨，我的鼻子前边干脆直接塌下来了。我是高鼻梁，想象一下一个人的鼻子从上到下突然变高然后又变低，喂喂喂，这是哪片山区吗？侧脸我就是伏地魔本魔好吧。
就像别人说的，容貌焦虑那是给建模还行的人的，俺这种丑的根本不用焦虑。好嘛，破罐破摔了，不管它了。不过网上刷到跟我脸型像还好看的小姐姐真的好羡慕啊。哎算了，还有AI和美颜可以让我偶尔对着自己臭美一下。
	
而我的出身和原生家庭是我自己不能决定的。我曾经想过，为什么有的人一出生就可以拥有我一生都可能得不到的东西，财富，阶层，见识，以及很多的试错机会。引用最近的热词“斩杀线”，我就在斩杀线上面的位置，距离只有一点点。那我自然一丁点都不敢动，生怕某些个错误的决定会让我直接被杀掉。
这种想法让我内耗过很长一段时间。不过后来我明白了，世界的本质就是各种不公平，如果我因为甚至不是我自身的原因就在原地踌躇，那我也没办法走得更远了。顺便提一句，难道因为我生下来就是笨蛋，就不配过自己的生活了吗？没有的东西可以用各种方法弥补，没有财富就上班积累，没有知识就阅读学习，没有见识就去多看看外面的世界。每个人的世界都是不一样的，我只希望能在自己的世界里找到一处安身之所。


### 3. 这一年中取得了哪些成长和进步，有做令自己满意的事吗？

答：
了解自己的现状也算是一种进步，毕竟之前对自己的认识都是不太客观的。还有学到了很多知识，获得了做许多事的丰富经验经历，具体就不一一列举了。
	
至于令自己满意的事，那可要说道说道了。
最令我满意的就是做饭！没有人教，我只能从各种做饭视频中学习。期间做出了非常多非常多的黑暗料理，最难我的就是火候和一些食材的处理，还有调味。糊锅是常有的事，然而糊过几次之后就能明白问题出在哪里，比如有些香料容易糊，淀粉沉底会糊。还有调味，学到一个口诀，"调和五味盐为首，酸辣苦甜醋在先"，至于量是多少，就靠自己的舌头和经验了。很感谢自己的舌头能吃得下很多奇怪的东西，给了我很多试错的自信（就算做坏了大概率自己还是吃的下去的）。
其次便是能在自己感兴趣的方面做出一些事，虽然微不足道，但我已经很开心了。比如为几个仓库提了PR，自己写了一些小工具（写毕设和工作用的），用AI完成了现在这个博客（虽然很丑）。接下来就继续加油！
最后就是，我终于敢在大多数人面前做好自己，表达自己真实的感受和想法，做决定时也不怎么会受外界干扰了。对自己来说这是一个非常大的进步，接下来继续加油啦！


### 4. 经历了哪些令你印象深刻的事？

答：
好多事都令我难忘，现在脑子有点乱，挑几件记得很清晰的大致说一说吧。
首先是跟身边的各种人出柜自己的性别认同。我以为我身边的很多人对这方面的态度是很保守的，可是在我坦白后他们却出乎意料地表示能够接受，甚至鼓励我去寻找真实的自己。身边的人包括我的发小，各个阶段的亲朋好友，部分家人和一些同学。今年很多的感动时刻就来源于此了。
然后是在做毕业设计期间跟导师的互动。我们只在开题，初审和终稿时聊过天，见面次数不超过3次，微信聊天不超过2页，然后竟然神奇地顺利毕业了。只能说本科论文就是很水啦。
然后就是，我今年最勇敢的一瞬间。我在大一时认识一位同学，从见她第一面时就被她的气质吸引，我想与她辩论，想知道她的想法，想了解她对各种事物的反应。我不知道这该是哪种感情，她在很多方面太像我心中的理想人了，我想与她交流，想变得像她一样好。后面的一段时间我们之间的交流机会不是很多了，将要毕业时我们已近半年没有联系，我知道今后我们的道路基本不会再交汇。我很怯懦不知如何开口，但最终还是鼓起勇气要了一张跟她的合影。这是我毕业时唯一一次主动跟别人合影。
之后就有很多事情了，在家时的骑行，找工作租房时遇到的蟑螂，在出租屋应聘两个月遇到的各种奇葩公司，以及在此期间受到的好朋友们的关心和照顾。
有趣的事情实在太多太多，我想我永远不会忘记它们的。


### 5. 对新的一年有做规划吗？

答：
我只希望自己能比2025年做得更好，除此之外就没什么规划了。不过有一些想做的尝试：
开始读书，每周都要给自己读书的时间。
重新学一下数学，和费曼讲义，只是因为感兴趣。
积累一些资金尝试投资，钱生钱（真的很需要钱，穷怕了OwO）。
继续做一些可能有点用的小项目。
继续尝试减肥，你瘦下来绝对比现在好看！

<script>
(function(){
  function initFireworks() {
    const c = document.getElementById('fireworks-canvas');
    if(!c) {
      setTimeout(initFireworks, 100);
      return;
    }
    
    const ctx = c.getContext('2d');
    if (!ctx) return;
    
    function resize(){
      const dpr = window.devicePixelRatio || 1;
      c.width = Math.floor(c.clientWidth * dpr);
      c.height = Math.floor(c.clientHeight * dpr);
      ctx.setTransform(dpr,0,0,dpr,0,0);
    }
    window.addEventListener('resize', resize);
    resize();

    const rockets = [];
    const particles = [];
    function rand(a,b){ return a + Math.random()*(b-a); }

    class Rocket{
      constructor(){
        this.x = rand(0, c.clientWidth);
        this.y = c.clientHeight;
        this.vx = rand(-1,1);
        this.vy = rand(-11,-8);
        this.color = `hsl(${Math.floor(rand(0,360))} 90% 60%)`;
        this.exploded = false;
      }
      update(){
        this.vy += 0.15;
        this.x += this.vx;
        this.y += this.vy;
        if(this.vy >= 0 && !this.exploded){ 
          this.explode(); 
          this.exploded = true; 
        }
      }
      draw(){
        ctx.beginPath(); 
        ctx.fillStyle = this.color; 
        ctx.arc(this.x, this.y, 2, 0, Math.PI*2); 
        ctx.fill();
      }
      explode(){
        const n = rand(30,60)|0;
        for(let i=0;i<n;i++){
          const a = Math.random()*Math.PI*2;
          const s = rand(1,6);
          particles.push(new Particle(this.x,this.y,Math.cos(a)*s,Math.sin(a)*s,this.color));
        }
      }
    }

    class Particle{
      constructor(x,y,vx,vy,color){
        this.x = x; 
        this.y = y; 
        this.vx = vx; 
        this.vy = vy; 
        this.alpha = 1; 
        this.color = color; 
        this.size = rand(1,3);
      }
      update(){ 
        this.vy += 0.08; 
        this.vx *= 0.98; 
        this.vy *= 0.98; 
        this.x += this.vx; 
        this.y += this.vy; 
        this.alpha -= 0.01; 
      }
      draw(){ 
        ctx.beginPath(); 
        ctx.fillStyle = this.color; 
        ctx.globalAlpha = Math.max(0,this.alpha); 
        ctx.arc(this.x,this.y,this.size,0,Math.PI*2); 
        ctx.fill(); 
        ctx.globalAlpha = 1; 
      }
    }

    function loop(){
      ctx.fillStyle = 'rgba(0,0,0,0.1)';
      ctx.fillRect(0,0,c.clientWidth,c.clientHeight);
      
      if(Math.random() < 0.05) rockets.push(new Rocket());
      
      for(let i=rockets.length-1;i>=0;i--){ 
        rockets[i].update(); 
        rockets[i].draw(); 
        if(rockets[i].exploded) rockets.splice(i,1); 
      }
      
      for(let i=particles.length-1;i>=0;i--){ 
        particles[i].update(); 
        particles[i].draw(); 
        if(particles[i].alpha <= 0) particles.splice(i,1); 
      }
      
      requestAnimationFrame(loop);
    }
    
    loop();
  }
  
  // 启动初始化
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initFireworks);
  } else {
    initFireworks();
  }
})();
</script>
