## 菜谱

先来记录下每道菜需要的材料和制作步骤。


### 东坡肉
这次做个只用高压锅版本的。

材料：
- 五花肉
- 小葱、姜
- 黄酒，冰糖，酱油，老抽
步骤：
1.  猪五花肉需要刮干净表皮，处理完后凉水下锅焯水至快煮沸定型，清洗后切成大方块。
2. 小葱十几根放在高压锅底，姜切四五片放在上面。放入切好的五花肉，肉皮朝下（形状整齐）。
3. 倒入黄酒半斤多，放入十几块冰糖，加入一炒勺酱油，老抽1/3炒勺。五六跟小葱打结，做三束，盖在肉顶部。
4. 加水直至没过五花肉，高压锅上压后约半小时即可。出锅后倒入炒锅进行收汁上色即可。


### 五花肉油豆腐

材料：
- 五花肉，油豆腐，金针菇，青红辣椒
- 蒜，盐，味精
- 料酒，生抽，老抽，耗油，烧烤料（辣椒粉），土豆淀粉

步骤：
1. 大蒜切末，青红辣椒切小块（可加甜椒），金针菇洗净备用。准备些许水淀粉。
2. 五花肉切小块后放入高压锅中直接煸炒，出油后按口味倒掉多余的猪油。之后倒入蒜末翻炒出香味，再加入金针菇煸炒出水分。
3. 加入料酒，生抽，老抽，冰糖，耗油，烧烤料，翻炒均匀，用量根据口味。
4. 油豆腐倒入高压锅中，加水没过油豆腐，高压锅上气后约五分钟即可。开盖后倒入青红辣椒，大火翻炒直至收汁即可。

### 排骨炖豆角

材料：
- 排骨，油豆角。
- 八角（一斤排骨一个），花椒粉，葱，姜，蒜。
- 黄酒或料酒，酱油，盐，老抽。

步骤：
1. 豆角常规处理后，锅中倒油，煸炒至翠绿色。
2. 锅中倒油，放入八角炸出香味，放入大段葱姜，之后放入排骨直接煸。排骨煸透之后撒入花椒面约两勺。然后倒入黄酒一炒勺，酱油一炒勺，染色后加入开水没过排骨。之后煎煮打沫，水开后中小火盖盖炖煮约半小时即可。
3. 滤除汤汁中的香料，大火炖五分钟左右，逐渐撒入盐调味，五分钟后在上面盖上豆角，不需要盖盖。逐渐收汁，待粘稠后加入少量老抽调色，拍几瓣蒜放在里面搅拌均匀。


### 扒白菜卷

材料： 
- 白菜 1颗，猪肉（肥瘦肉二八分） 约 1斤 ，海米（可选，可换虾仁）
- 大葱、姜、鸡蛋（蛋清）
- 料酒、盐、味精、香油、胡椒粉、淀粉

步骤：
1. 白菜切掉白菜梆，白菜叶用开水烫一下烫软，以便卷馅。烫完后晾凉，用纸沾掉菜叶上的水，不然馅容易滑。
2. 猪肉切成末，葱姜切末，海米少许切末，混合后开始搅拌。顺、逆时针交替搅拌使充分混合，期间分多次加瓶盖量的水（可以使口感更嫩）。搅拌途中加鸡蛋清，料酒，盐，味精，香油，胡椒粉。馅料充分混合且出香即可。
3. 每张白菜叶填入适当的馅，从白菜梆处向叶尖开始卷，卷紧一些防止散开，卷成圆柱，完成后逐个放入盘中。
4. 烧水，水开后开始蒸7分钟左右。蒸完后盘子里剩下的汤水倒入碗里，最后勾芡使用。将蒸好的白菜切成长宽高相近的小块，摆入盘中。准备半小碗水淀粉。
5. 留下的汤水倒入锅里，加半炒勺水，加入盐、味精、水淀粉，中火逐渐勾芡至略粘稠，最后倒入香油。做好的芡汁浇在摆好的白菜卷上即可。


### 松鼠鳜鱼

材料：
- 桂鱼。
- 冬笋，冬菇，胡萝卜，青豆。
- 葱，姜，蒜。
- 黄酒，料酒，盐，糖，番茄酱，胡椒粉，白醋，淀粉（玉米淀粉）。

步骤：
1. 完成最难的改刀步骤！可以自己参考各个师傅的做法，我学的是 [老饭骨-松鼠桂鱼](https://www.bilibili.com/video/BV1rm4y1e73K) 。
2. 改好的鱼肉放入碗中开始腌制。倒入两勺黄酒。加入两小勺盐撒均匀进行腌制。
3. 冬笋，冬菇，胡萝卜切等量小丁热水焯一下，准备等量青豆（约十几粒即可）单独焯一下（方便过凉水）。葱姜蒜切小粒。
4. 炸鱼。使用淀粉将其抹匀，整条鱼的可见的外表面都抹上。油温200度下锅炸，注意先提住上端和尾巴，炸鱼身使其弯折定形后再整只放入。炸完定形后捞出，油温到240度以上后再快速复炸一遍使其酥脆。炸完后可放入盘中摆盘。
5. 炒汁：一锅烧热两三勺油待泼热油，另一锅中少许油，依次慢炒葱姜蒜，番茄酱，加入少许水后放入糖，根据口味调制。加入三碗开水，倒入半炒勺白醋，料酒，一勺胡椒粉。加入之前炒好的蔬菜粒，加入一炒勺淀粉水后煮至缓慢冒泡，最后倒入一两勺热油。炒至粘稠。
6. 将炒好的汁均匀浇在炸好的鱼上。



### 虫草花清汤狮子头

材料：
- 五花肉（选取瘦肉偏多的部分，不要猪皮）约2斤，荸荠（马蹄）10粒（占1/4），虫草花 一大把，清鸡汤（没有可以使用鸡汁加水......）
- 大葱，姜，鸡蛋
- 盐，胡椒粉，黄酒，玉米淀粉

步骤：
1. 五花肉提前冷藏方便切块。五花肉切成筷子头大小的丁。且完后再粗剁一下（大概三四十刀），让肉开口方便粘在一起。拍开荸荠，不要太烂，同样粗剁一下。然后分别放在不同的碗里。
2. 大葱和姜各切四五段小块，拍碎放入另一个碗里，加一炒勺水，倒入等量黄酒，撒些许胡椒粉。抓匀浸泡制作葱姜水。
3.  三个鸡蛋打入放肉的碗中（可以只用蛋清来让狮子头更白），撒入一两玉米淀粉（三四把），然后开始抓拌。抓拌手法是搅一下抓起然后摔下（如果搅拌最终就不会有很多弹性），直至有些许粘度时倒入一些葱姜水，葱姜水分三次加入。期间分多次加盐，一次一勺（水多加盐，盐多加水，保持适当粘稠度）。
4. 肉的粘稠度足够后倒入荸荠，抓拌均匀，此时摔的力度可以轻一些。持续抓拌直到团起来能成团，轻轻一捏也能散的程度。
5. 制作芡汁，加入玉米淀粉和一个鸡蛋清，加一些水使其不要太稠，可以拿漏勺过过滤一下。
6. 烧一锅水（中小火，不要沸腾），团狮子头，在左右手里反复翻滚团成一颗球（用大约一个小蘸料碗大小的馅），拿一只手沾点制作好的芡汁抹在上面，直到成形不散时轻下入锅里。
7. 保持文火，期间撇去表面的浮沫。切大段葱段和姜片下入，倒入黄酒（多一点）。根据口味再加入若干勺盐，加少许糖补鲜味，加入胡椒粉去腥。烧开后炖两个小时，不要盖住（散掉一些味道）。
8. 虫草花蒸约5分钟，也可使用鸡（汁）汤煮一下，煮后捞出。将鸡（汁）汤煮开。
9. 炖完后捞出狮子头（不要使用原汤，那个太咸），装入汤碗中，倒入煮好的鸡汤，在狮子头间的间隙撒上虫草花装饰即可。

### 鱼香鸡扒

材料：
- 鸡腿肉（只取大腿端） 三四块
- 洋葱，胡萝卜，柠檬，芹菜，香叶
- 盐，白糖，黑胡椒，白胡椒粉，葱，姜，蒜
- 鸡汤，老抽，黄酒，黑醋，玉米淀粉，面粉

步骤：
1. 鸡腿肉处理好后铺平，无皮的一面朝上，菜刀轻砍，将肉剁碎但不能剁开鸡皮。肉比较细碎后，拿菜刀下端点戳几个洞，防止鸡皮收缩过度。
2. 将鸡肉放入碗中，之后加入1/4个洋葱，四五片胡萝卜。三四片柠檬，四五段芹菜叶，三四片香叶，5克盐，倒入一炒勺黄酒，一些和胡椒。抓拌均匀，然后腌制20分钟。
3. 葱姜蒜等比例切末，放入普通菜盘约占3/4。泡辣椒与豆瓣酱2比1填满盘子剩下的1/4。 
4. 准备料汁。一碗鸡汤（或加入一些鸡汁的一碗水），半勺老抽，两勺黄酒，一克盐，三勺白糖，三勺黑醋，两小勺白胡椒粉。玉米淀粉制作水淀粉备用。
5. 锅里烧中等量油，油热倒入葱姜蒜煸小火炒出香味，之后倒入豆瓣酱和泡辣椒。炒香后倒入料汁煮开，随后加入淀粉水勾芡至稍粘稠。
6. 准备一碟面粉，将研制好的鸡腿肉取出，去掉腌料后裹上面粉。锅中倒入少许油，将鸡排放入煎炸，两面煎至金黄且筷子可以轻易穿透时即可。
7. 煎完后改刀至条状。但仍维持鸡排形状，然后摆盘。将最后使用料汁浇盖即可。

其实可以用其他县现成的酱料代替，比如叉烧酱番茄酱，自己只烧个鸡扒就可以，味道已经够好了。

### 油焖大虾

材料：
- 大虾（黑虎虾） 十几只
- 油
- 花椒，姜，大葱（比例为 1:2:6），黄酒，盐，冰糖
- 剪刀✂

步骤：
1.  拿剪刀剪去虾头（一般是剪到虾枪根部，这里用虾头做下一道菜），挑去虾头部位的虾胃，剪掉虾须，尾巴剪齐，剪去尾尖。从第二节背部铲开，挑去虾线。
2. 炸葱椒油。准备比例为 1:2:6的花椒，姜，大葱，再切些姜米。锅中烧油，三四成油温（100度左右）放入花椒，油热离火，炸出香味后下入姜，最后下葱，保持中火慢炸。葱变黄且炸出香味后过滤。整个葱椒油可以留下炸其他的菜。
3. 煎虾，中火烧油（油比平常煎稍多），下入虾，一面变色后翻面。若留有部分虾头，可以挤压虾头挤出虾油。
4. 煎至双面金黄且后下入姜米，倒入半炒勺黄酒，加2炒勺水，3克盐，6块左右小冰糖。大火烧开后盖锅盖，小火焖15分钟。之后捞出摆盘
5. 过滤剩下的汤汁，滤掉姜米，汤汁重新入锅大火收汁，收浓后加入一点葱椒油，浇盖在虾上。

### 虾头烧豆腐

材料：
- 豆腐，熟鸡蛋黄 2颗（没有鸭蛋QwQ），胡萝卜 1根
- 油（之前剩下的葱姜油也可）
- 大葱，姜，小葱（装饰）
- 胡椒粉，黄酒（料酒也行），番茄沙司（根据个人口味选择），盐，糖，味精，土豆淀粉

步骤：
1. 豆腐两面老皮切掉，之后切成方形的豆腐小块。切完后凉水下锅，小火慢煮，加入几小勺盐（使嫩弹，加底味）。煮至火快开时关火浸泡。
2. 烧油，各切四五大片姜和大葱，大葱备用，先放入姜片，煸出香味后下入虾头，开始拍打挤压虾头，挤出虾油，中火持续边煸边按。基本煸出大部分虾油后下入葱（早下容易焦）。
3. 加入较多胡椒粉去腥，半勺黄酒沿锅边转圈倒入。一分钟后根据准备的豆腐的量加入开水，熬至烧开后小火浸泡，使虾油充分混合，最后过滤掉所有大料留下虾汤。
4. 鸡蛋黄拿刀抹开，胡萝卜切成条（不要芯）拍碎后切细，切一片姜并切成末。锅洗净后烧少许油，先下入姜末煸出香味后下入蛋黄和胡萝卜（根据口味添加番茄沙司），同样挤压拍打，煸炒出红色后倒入虾汤，煮开后下入豆腐。
5. 制作水淀粉，汤汁粘稠后加入淀粉开始收汁，这个汁要浓稠一点，因为豆腐的味道基本靠汁。最后倒入盘中，撒入小葱花装饰即可。


### 下饭鸡蛋炸酱

材料：
- 青椒 3根，红辣椒 1根，鸡蛋 4颗，黄瓜一根
- 大葱
- 黄豆酱，耗油，生抽，老抽，黑醋，白糖（冰糖或白砂糖可替），十三香，鸡精

步骤：
1. 青椒切成小块，红辣椒切成小圈，大葱切葱花，黄瓜切丝，放入一个碗中。
2. 制作酱料。黄豆酱4勺，耗油1勺，生抽1勺，老抽半勺，黑醋半勺，一小勺白糖，一小勺十三香，一勺鸡精。
3. 鸡蛋打完后搅拌均匀，烧油至冒烟后转为小火，将鸡蛋液小流量缓慢倒入油中并不断进行搅拌。
4. 鸡蛋炸熟后下入前面切的菜，大火翻炒，辣椒香味出来后倒入酱汁，翻炒约1分钟，倒入约2炒勺水继续翻炒直至收汁即可（收汁不需要太浓稠，这是道下米饭的菜）。
